📊 Projeto SQL de Vendas

Este projeto simula um banco de dados de vendas de uma loja fictícia e demonstra como um analista de dados pode usar **SQL** para extrair insights de negócio.  
O objetivo é praticar consultas SQL e mostrar exemplos de análises que poderiam alimentar dashboards em ferramentas como Power BI ou Tableau.

---

🔹 Estrutura do Banco de Dados

O banco é composto pelas seguintes tabelas:

- Clientes: informações dos clientes (ID, nome, cidade, estado).
- Produtos: catálogo de produtos (ID, nome, categoria, preço).
- Pedidos: registros de compras realizadas (ID, cliente, data, valor total).
- Itens_Pedido: detalhes dos produtos em cada pedido (ID, pedido, produto, quantidade, preço unitário).

---
 🔹 Consultas Implementadas

Exemplos de análises disponíveis neste projeto:

1. Faturamento mensal → soma das vendas por mês.
2. Ticket médio por cliente → valor médio gasto por cliente.
3. Top 5 produtos mais vendidos → ranking dos produtos mais populares.
4. Distribuição de vendas por categoria → participação de cada categoria no faturamento.
5. Clientes mais ativos → clientes com maior número de pedidos.
