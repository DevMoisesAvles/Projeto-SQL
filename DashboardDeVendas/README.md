

## 📊 Visão Geral

O **NexStore Analytics** é um dashboard de vendas desenvolvido para simular um ambiente real de análise de dados em e-commerce.

O projeto apresenta métricas estratégicas, visualizações interativas e insights de negócio com foco em **tomada de decisão orientada a dados (data-driven)**.

---

## 🎯 Objetivo

Demonstrar habilidades práticas em:

- Análise de dados aplicada ao negócio
- Construção de dashboards interativos
- Visualização de métricas (KPIs)
- Desenvolvimento front-end com foco em UI/UX
- Comunicação de insights

---

## 🧠 Insights de Negócio

- Crescimento de receita ao longo dos meses
- Relação entre volume de pedidos e faturamento
- Canais de aquisição mais eficientes
- Produtos com maior impacto financeiro
- Regiões mais lucrativas
- Taxa de conversão do funil de vendas
